Core team

Abhas Kumar(MD)

Graduated from IITK, major in Computer Science. Have 5 years work experience in Intel. He at Intel, designed tools for next generation Xeon and other server grade processors. He is an accomplished engineer with multiple publications in his name in many reputed journals. He has also published multiple papers inside Intel regarding innovative tools for server validation. Worked with other startups like Innovel Technologies, Srapps Consultancy etc.

Akshay Mathur(Director)

Graduated from IITK, and worked for 5 years at Aurora Integrated Systems, a pioneer in Indian UAV (Unmanned Aerial Vehicles) space (now a Tata Group Company). Developed Ground Control Station (GCS), a software suite to control the UAV completely, Base Station (an embedded system relaying commands between GCS & UAV) and some parts of AutoPilot for the UAV. After this, have created an Android based Car Infotainment System which can be installed in all 2DIN cars in India. Other works include developing Remote Monitoring & Control System for solar systems, CAN based OBD2 reader, writing drivers of various components (touch screen, radio, IR remote, FM radio, etc.) for Android, etc.

Harish Chandra Singh(Director)

Graduated from NIT, Patna in Electronics and Electrical Engineering. Has worked with Alstom as system architect for power and communication based automatic control systems involving SIL4 level safety critical technologies. He has designed both embedded control devices as well power electronics systems. Latest projects involve automatic train control systems for Metros, High speed data logger and electronic detonator system for DRDO, Control system for E-bike and alcohol vending machine, Power electronics and communication system for POS system involving high speed power path selection.
